## 光刻工艺
### 光刻流程：
    掩模版的制造
    1.表面清洗：洗去表面杂质离子
    2.涂胶、匀胶（转速、胶量、类型）
    3.前烘（挥发溶剂部分，使其坚硬耐腐蚀，获得机械性能）
    4.通过掩模版光刻（露出光刻窗口，正交光分解图案与掩模版一致、负胶光交联图案与掩模版相反）
    5.后烘（激活感光分子，弥补曝光不充分的部分，提高曝光质量）
    6.显影（用有机溶剂洗去部分光刻胶，需注意浸润时间，防止过度显影）
    7.图形检查（检查图形完整性，如果不满足要求则重新开始光刻流程）
